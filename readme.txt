A simple plugin to enable admins/editors to leave notes on bbPress topic replies.

== Description ==

This is a simple extension that allows forum admins and editors to leave notes on topic replies. If you run a support forum, or any other community forum, this is a great way for forum moderators to communicate privately between each other in relation to the replies.

== Installation ==

1. Activate the plugin
2. Bee happy
